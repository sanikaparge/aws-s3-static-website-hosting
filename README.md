# AWS S3 Static Website Hosting

## Project Overview
This project demonstrates hosting a static website using Amazon S3.

## Technologies Used
- AWS S3
- HTML
- CSS
- Linux Basics
- Cloud Computing

## Project Architecture

User Browser
      ↓
AWS S3 Bucket
      ↓
Static Website Hosting
      ↓
Website Accessible via Public URL

## Implementation Steps

1. Created an S3 Bucket
2. Uploaded website files
3. Configured bucket permissions
4. Disabled Block Public Access
5. Added Bucket Policy
6. Enabled Static Website Hosting
7. Accessed website through the S3 endpoint URL

## Features
- Static Website Hosting
- Public Access Configuration
- Bucket Policy Management
- Simple and Scalable Deployment

- Understanding AWS S3
